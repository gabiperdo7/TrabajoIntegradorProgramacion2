# Food Store - Trabajo Practico Integrador Programacion 2

## Requisitos

* Java 21 o superior
* Apache NetBeans (recomendado)

## Ejecucion del proyecto

1. Clonar o descargar el repositorio.
2. Abrir el proyecto en Apache NetBeans.
3. Ejecutar la clase `Main`.
4. Utilizar el menu de consola para gestionar:

   * Categorias
   * Productos
   * Usuarios
   * Pedidos

## Observaciones

* El sistema funciona completamente por consola.
* Los datos se almacenan en memoria mediante colecciones.
* No requiere configuracion adicional ni base de datos.
